# 📊 Student AI Usage Analytics Dashboard

## Overview

This project analyzes how students use Artificial Intelligence (AI) tools in their academic activities. The dashboard was developed in Microsoft Excel to provide a clear view of student demographics, AI adoption, study behavior, and academic performance.

The project combines descriptive analytics, KPI reporting, and interactive dashboard design to support educational data analysis.

---

## Project Objectives

The dashboard answers the following questions:

- How many students participated?
- What are the student age distributions?
- What education levels are represented?
- How many students use AI?
- Which AI tools are most popular?
- What are the common purposes of AI usage?
- How do study habits vary?
- How do grades compare before and after AI usage?
